## What to include in bug reports
- Platform (iOS / iPadOS / Android)
- Deskless version (Settings/About)
- OS version (e.g., iOS 26.1)
- Exact steps to reproduce
- What you expected vs what happened
- Screenshots/logs (please **redact** personal data, tokens, repo URLs, etc.)
